# Frontend - LinkedIn Clone (React)

## Setup
1. Copy `.env.example` to `.env` and set `REACT_APP_API_URL` to your backend URL (e.g., https://your-backend.onrender.com).
2. Install dependencies: `npm install`
3. Run: `npm start`

## Notes
- The app stores JWT token and user in localStorage.
- Designed to be deployed on Netlify or Vercel. Set `REACT_APP_API_URL` environment variable in deployment settings.
